## 课程绪论

关于动态内存管理：

```c++
int* x = new int(2); // 动态分配单个 int 变量，初始化值为 2
delete x;

int* x = new int[2]; // 动态分配长度为 2 的 int 数组
cout << x << endl << *x << endl;
delete[] x;
```

Data element：一条数据，可以包含多个内容，比如学生信息；
Data object：多条相同属性的数据，多个学生信息；
除非要求，否则不需要很明确的划分

ADT，可以理解为一个容器。包含关于这个数据的信息和其操作。

ADT = （D，R，O），D: Data Object R: Relationship O: Operations

算法表示符号（Landau Symbol），符号之间的关系，经典的图。

1. $f(n) = o(g(n)) \quad \lim_{n \to \infty} \frac{f(n)}{g(n)} = 0$ 
2. $f(n) = O(g(n)) \quad \lim_{n \to \infty} \frac{f(n)}{g(n)} < \infty$ 
3. $f(n) = \Theta(g(n)) \quad 0 < \lim_{n \to \infty} \frac{f(n)}{g(n)} < \infty$ 
4. $f(n) = \Omega(g(n)) \quad \lim_{n \to \infty} \frac{f(n)}{g(n)} > 0$ 
5. $f(n) = \omega(g(n)) \quad \lim_{n \to \infty} \frac{f(n)}{g(n)} = \infty$ 

## 链表

链表的ADT（Abstract Data Type）是 List，一种线性结构。

对于这种线性存储结构有两种实现形式：
- 数组
- 链表
对应有四种最基本的操作，即 **增（Insert）删（Delete）改（Retrieve）查（Find）**

### 数组实现形式 

*P25-36* 
在数组中的查找很好实现，遍历然后比较即可。

对于插入元素来说，需要移动**插入位置后面的元素**，第 $i$ 个（ $i$ 为索引）元素之前插入，需要移动后面 $n-i$ 个元素。

移除元素，需要移动移除位置后面的元素，第 $i$ 个移除，移动 $n-i-1$ 个元素。

对于插入、移除和查找这三个操作，使用数组实现的时间复杂度均为 $O(n)$ ，空间复杂度为 $O(1)$。
